La propiedad CSS display controla cómo se muestra un elemento en la página. Los valores más comunes son:

  • block: El elemento ocupa todo el ancho disponible, comenzando en una nueva línea.
  • inline: El elemento ocupa solo el espacio necesario y se muestra en línea con otros elementos.
  • inline-block: Combina características de block e inline; el elemento puede tener un ancho y alto definidos, pero se muestra en línea.
  • none: El elemento no se muestra en la página y no ocupa espacio.
